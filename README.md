# TestContactsApp
Test application for storing, adding and deleting contacts

# Used technologies
- MVVM Architecture
- Kotlin
- Android Architecture Components (Viewmodel, LiveData)
- Databinding
- Kotlin Coroutines
- Room
- Lottie
